# @neatui/navs

MCUI导航组件，基于MCUI核心@neatui/core, 官网 https://mcui.fekit.cn/

## 安装组件

```
npm i @neatui/navs
```

或

```
yarn add @neatui/navs
```

## 可选主题

- navs@a.scss
- navs@b.scss

## 使用示例

```html
<ul ui-navs="@a">
  <li class="active"><span>导航一</span></li>
  <li><span>导航二</span></li>
  <li><span>导航三</span></li>
</ul>
```
